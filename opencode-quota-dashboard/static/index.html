目前顯示的是 Demo 資料。若要監控 OpenCode Go，請先設定：

# PowerShell
$env:OPENCODE_GO_WORKSPACE_ID = "你的 Workspace ID"
$env:OPENCODE_GO_AUTH_COOKIE  = "你的 auth cookie"

# 然後重開 server
node server.js

取得方式：登入 opencode.ai → 開 DevTools → Application → Cookies → 複製 auth 值與 URL 中的 workspace ID。
